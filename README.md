# Layoffs Data Cleaning Project (SQL)

## Project Overview
This project focuses on cleaning and preparing a real-world layoffs dataset using MySQL.
The raw data contained duplicates, inconsistent values, NULLs, and incorrect date formats.

## Objectives
- Standardize text and date formats
- Handle NULL values responsibly
- Remove duplicate records safely
- Prepare the dataset for analysis

## Cleaning Steps
1. Created a staging table to protect raw data
2. Standardized company names, industries, countries, and dates
3. Converted empty strings to NULL values
4. Handled NULLs based on business logic
5. Removed duplicate rows using ROW_NUMBER()

## Tools Used
- MySQL
- SQL Window Functions

## Result
The dataset is now clean, consistent, and analysis-ready.
